erase_flash(connect=True)[source]¶

Erase program memory flash from FuelSensor device. This function only clear the app. Bootloader persist on memory.

‘connect’ parameter controls if tcp connection is stablished before executing the the erase flash operation. Always keep it as True, or wich is the same, call the function with the default values as erase_flash()
